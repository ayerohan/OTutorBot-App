# OTutorBot 🤖📚

OTutorBot is an intelligent online learning assistant designed to help students understand concepts, practice effectively, and prepare for exams. It leverages modern AI and automation tools to provide a personalized and interactive learning experience.

---

## 🚀 Features

- 🎯 **AI-Powered Learning Assistance**  
  Uses the Gemini API to provide explanations, answer questions, and guide users through various topics.

- 📝 **Exam Preparation System**  
  Automatically generates practice tests using the Google Forms API to simulate real exam conditions.

- ⚡ **Real-Time Doubt Solving**  
  Users can ask questions and receive instant, structured responses.

- 📖 **Personalized Learning Experience**  
  Adapts responses based on user queries and learning needs.

---

## 🛠️ Tech Stack

- **Backend**: Spring Boot (Java)
- **APIs Used**:
  - Gemini API (AI-based learning assistance)
  - Google Forms API (exam generation)
- **Frontend**: HTML (Templates)
- **Build Tool**: Maven
